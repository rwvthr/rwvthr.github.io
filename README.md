## CalcNexus

### 📍 Geometria Analítica

Este repositório tem o intuito de ajudar no estudo de Geometria Analítica, servindo como material de apoio para a compreensão de conceitos fundamentais, fórmulas e aplicações práticas da disciplina.

### 🎯 Objetivos:

- Facilitar o aprendizado de tópicos como vetores, retas e planos.

- Oferecer visualizações ou exemplos que auxiliem na fixação do conteúdo.

- Servir de guia rápido para consulta de fórmulas e propriedades.
